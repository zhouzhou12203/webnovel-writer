 
# Web 前端界面功能说明

## 概述

为 Webnovel Writer 创作系统提供可视化 Web 界面，基于 Vite + Vue 3 + FastAPI 技术栈，集成 AI 自动写作功能。

---

## 技术架构

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | Vue 3 + Vite | 响应式单页应用 |
| 状态管理 | Pinia | 全局状态管理 |
| HTTP 客户端 | Axios | API 请求封装 |
| 路由 | Vue Router 4 | SPA 路由管理 |
| 后端 | FastAPI | RESTful API 服务 |
| AI 服务 | OpenAI 兼容 API | 默认 http://localhost:8317 |
| 端口 | 前端 5173 / 后端 8080 | 默认端口配置 |

---

## 启动方式

### 1. 启动后端
```bash
cd backend
python3 -m uvicorn main:app --host 0.0.0.0 --port 8080 --reload
# python3 -m uvicorn main:app --host 0.0.0.0 --port 8080
```

### 2. 启动前端
```bash
cd frontend
npm install  # 首次运行
npm run dev
```

### 3. 访问界面
打开浏览器访问：http://localhost:5173

## 功能页面

### 首页 (/)
- 项目统计：章节数、总字数、题材类型
- 快捷操作入口
- 核心功能介绍

### 项目管理 (/project)
- 查看项目状态
- 初始化新项目（选择题材、模式）
- 配置 API 密钥

### 大纲编辑 (/outline)
- 总纲编辑
- 卷大纲管理
- 新建卷大纲

### 章节创作 (/write)
- 章节列表管理
- Markdown 编辑器
- 实时字数统计
- 保存章节

### 实体管理 (/entities)
- 按类型筛选（角色/地点/物品/伏笔）
- 实体详情查看
- 搜索功能

### RAG 检索 (/rag)
- 语义搜索测试
- 向量/BM25/混合检索模式
- 检索结果展示

## API 端点

| 模块 | 路径前缀 | 功能 |
|------|---------|------|
| 项目 | /api/projects | 状态、配置、初始化 |
| 大纲 | /api/outlines | 大纲 CRUD |
| 章节 | /api/chapters | 章节 CRUD、统计 |
| 实体 | /api/entities | 实体查询 |
| RAG | /api/rag | 检索测试 |

## 文件结构

```
backend/
├── main.py              # FastAPI 入口
├── requirements.txt     # Python 依赖
└── routers/             # API 路由
    ├── projects.py      # 项目管理
    ├── outlines.py      # 大纲管理
    ├── chapters.py      # 章节管理
    ├── entities.py      # 实体管理
    └── rag.py           # RAG 检索

frontend/
├── src/
│   ├── api/             # API 封装
│   ├── router/          # 路由配置
│   ├── stores/          # Pinia 状态
│   ├── views/           # 页面组件
│   ├── App.vue          # 主组件
│   ├── main.js          # 入口
│   └── style.css        # 全局样式
├── index.html
└── vite.config.js
```
